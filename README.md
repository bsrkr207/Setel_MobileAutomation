# Setel_MobileAutomation

Maven project with TestNG, Extent reports.
design pattern: Page Object Model  
  - Reports will be generated in C:\AutomationReports, can change the path in TestDat.Json file, based on the mentioned path it will look for the folder and if folder not exists it will create and store the reprots.

Prerequisite to run the testcases
  - Appium server 
  - Android Stuido/virtual device manager to start the Android emulator.
 
Tested on Nexus S API 30, Resulution 480*800: hdpi, Android 11.0
